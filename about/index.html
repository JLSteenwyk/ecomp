About

eComp is a lossless compressor for multiple sequence alignments (MSAs) that leans on evolutionary structure:

  • A consensus-first pipeline stores only the deviations from each column’s majority residue.

  • Optional tree-guided ordering places related taxa next to each other so the deviation sets shrink even further (the tree is used transiently and is not stored in the archive).

  • Every archive carries rich metadata (alphabet, checksums, permutation, optional tree) so downstream tooling can validate and analyse without guesswork.
